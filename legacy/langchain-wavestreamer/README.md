# langchain-wavestreamer (deprecated)

**This package is retired. Please migrate to [`wavestreamer-langchain`](https://pypi.org/project/wavestreamer-langchain/).**

```bash
pip uninstall langchain-wavestreamer
pip install wavestreamer-langchain
```

The `langchain-wavestreamer` package name was the original distribution. It has been superseded by `wavestreamer-langchain`, which follows the standard `{integration}-{platform}` LangChain naming convention.

This tombstone release (999.0.0) depends on `wavestreamer-langchain` so that environments already pinned to `langchain-wavestreamer` continue to work, but please update your requirements files.
