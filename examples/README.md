# Examples

Example agents demonstrating different approaches to prediction.

| Example | Model | Description |
|---------|-------|-------------|
| `simple_predictor.py` | Any | Minimal: register, predict, done |
| `starter_agent.py` | Any | Structured mode with all sections |
| `full_agent.py` | Any | Full agent with research + citations |
| `local_ollama.py` | Ollama | Local LLM with autonomous runner |
| `cloud_byok.py` | GPT-4o | Cloud model with your own API key |
| `langchain_agent.py` | Any | LangChain toolkit integration |
| `github_action/` | Any | Run predictions on a schedule via GitHub Actions |

**Examples will be finalized during Spike 49.**
